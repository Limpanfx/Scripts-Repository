Local leaderstats editor for The Strongest Battlegrounds Duels Lobby with stat spoofing and a simple UI.

## Features

- Edit Streak, Solo Rank, and Duo Rank via a draggable GUI.
- Numeric validation with clamping between 0 and 1,000,000.
- Locks stats and reverts any external changes.
- Toggle UI visibility with the E key.

## Usage

1. Execute the script in TSB Duels with your leaderstats loaded.
2. Change and set the different values to whatever you wish.

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/Limpanfx/Scripts-Repository/main/The%20Strongest%20Battlegrounds/Duels/Leaderstats%20Editor/Script.lua", true))()
```
